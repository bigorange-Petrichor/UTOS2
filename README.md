# A Unified Target-Oriented Sequence-to-Sequence Model for Emotion-Cause Pair Extraction

This repo contains the code of our TASLP'2021 paper:

A Unified Target-Oriented Sequence-to-Sequence Model for Emotion-Cause Pair Extraction

## Requirements

- Python 3.6
- PyTorch 1.8.0
- transformers 2.8.0
- pytorch_pretrained_bert 0.6.2

## Quick Start  

Run our model:

python src/main.py
